# ClearChat - AI聊天应用

## 项目简介

ClearChat 是一个基于 HarmonyOS 开发的AI聊天应用，支持多种AI模型提供商，提供流畅的对话体验和丰富的功能。

## 主要功能

### 🤖 AI对话
- 支持多种AI模型（OpenAI、Claude、Gemini等）
- 实时流式响应
- 对话历史管理
- 智能标题生成

### 🔧 模型管理
- 自定义AI提供商配置
- 多模型切换
- API密钥安全管理
- 连接测试功能

### 💬 对话管理
- 对话列表管理
- 消息编辑和删除
- 对话重命名
- 历史记录清理

### 🎨 用户界面
- 响应式设计，支持多种屏幕尺寸
- 侧边栏导航
- Markdown渲染支持
- 代码高亮显示

## 技术架构

### 核心技术栈
- **开发框架**: HarmonyOS ArkTS
- **UI框架**: ArkUI
- **数据库**: 关系型数据库 (RDB)
- **网络请求**: HTTP模块
- **状态管理**: ArkTS状态管理

### 项目结构
```
entry/src/main/ets/
├── components/          # 可复用组件
│   ├── MessageItem.ets     # 消息项组件
│   ├── MarkdownRenderer.ets # Markdown渲染器
│   ├── CodeBlock.ets       # 代码块组件
│   └── ...
├── pages/              # 页面组件
│   ├── Chat.ets           # 主聊天页面
│   ├── Settings.ets       # 设置页面
│   └── ModelSettings.ets  # 模型设置页面
├── services/           # 业务服务
│   ├── NetworkManager.ets    # 网络请求管理
│   ├── DatabaseManager.ets   # 数据库管理
│   ├── TitleGenerationService.ets # 标题生成服务
│   └── DataModels.ets        # 数据模型
└── utils/              # 工具类
    └── ArrayDataSource.ets   # 数组数据源
```

## 核心特性

### 🔒 安全性
- API密钥加密存储
- 输入内容安全验证
- 防止XSS和SQL注入攻击

### ⚡ 性能优化
- LazyForEach优化长列表渲染

- 网络请求错误处理优化

### 🎯 用户体验
- 智能对话价值判断
- 友好的错误提示信息
- 流畅的动画和交互
- 响应式布局适配

## 开发规范

### 代码规范
- 遵循ArkTS开发规范
- 使用TypeScript严格模式
- 组件命名采用大驼峰命名法
- 状态变量使用明确的前缀

### 性能规范
- 避免在build函数中进行复杂计算
- 合理使用@State、@Prop、@Link装饰器
- 优化ForEach和LazyForEach的使用
- 及时释放资源和清理监听器

### 安全规范
- 不在日志中输出敏感信息
- 对用户输入进行严格验证
- 使用安全的网络请求方式
- 定期更新依赖库版本

## 构建和运行

### 环境要求
- DevEco Studio 4.0+
- HarmonyOS SDK API 9+
- Node.js 16+

### 构建步骤
1. 克隆项目到本地
2. 使用DevEco Studio打开项目
3. 配置签名信息
4. 点击运行按钮或使用命令行构建

### 配置说明
- 在设置页面配置AI提供商信息
- 添加有效的API密钥
- 选择合适的AI模型
- 测试连接确保配置正确
